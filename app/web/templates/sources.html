Data sources

Every dataset behind this site, with the year it's from and a link to the original source. Vintages don't line up across sources — NYC publishes each on its own cadence — so we list each one explicitly. Most are refreshed when the relevant agency publishes a new release.

Backbone: nycschools by Adelphi Ed Tech

Most of the long-tail historical data on this site comes from the nycschools Python package (Matthew X. Curinga / Adelphi Ed Tech), which has been quietly assembling NYC DOE InfoHub releases, scraped DOE-portal data, and several supporting tables for years. It's AGPL-3.0, like this site. A fork of the package adds the NYSED Report Card Database loader (PR back to Adelphi pending) and the Guidance Counselor + Social Worker loader.

School demographics
2005 – 2024-25 · NYC DOE annual demographic snapshot — enrollment, race, ELL, SWD, poverty, ENI.
DOE per-school snapshots
~2016 (stale) · DOE per-school portal scrape — principal, admissions method, quality-review URL. Older than the rest; we still surface it where there's nothing newer.
3-8 ELA exams
2012 – 2022 (no 2019-20, 2020-21 — COVID) · NYS English Language Arts state tests.
3-8 math exams
2012 – 2022 (no 2019-20, 2020-21 — COVID) · NYS mathematics state tests.
Regents exams
2014 – 2022 · NYS Regents results, all subjects.
Class size
2022-23 · DOE InfoHub class-size report.
Pupil-to-teacher ratio
2022-23 · Same DOE InfoHub release as class size.
SHSAT outcomes
Multiple years · NYC Open Data SHSAT applicant + offer counts by sending school.
Galaxy budget
2022-23 · DOE Galaxy school-budget portal scrape.
High School Directory
AY 2021 · NYC Open Data HS directory dataset (8b6c-7uty) — admissions programs, AP courses, sports, transit.
School locations
Fiscal year 2019-20 · NYC Open Data school location points (resource wg9x-4ke6). Includes lat/lon, NTA, council district, BBL.

Direct from NYC DOE / Open Data

Datasets we pull directly (without going through the nycschools upstream), typically because they post-date the package's last release or aren't routed through it.

ES attendance zones
2024-25 · cmjf-yawu · Elementary school zone polygons for the address-based "where is this kid zoned?" lookup.
MS attendance zones
2024-25 · t26j-jbq7 · Middle school zone polygons. Districts 1, 7, 15 use choice-based admissions so they have no MS zones.
Co-Location Report
2020-21 (latest published; DOE has retired the series) · ugqd-j3dx · Maps each school to one or more building IDs; powers the "Co-located" list on every school page.
Guidance Counselor & Social Worker staffing
2025-26 (mid-year reporting) · DOE InfoHub · Per-school FTE counts and pupil-per-staff ratios. Updated annually.

NYSED School Report Card Database

All accountability tables under "Spending & staffing" / NYSED on school pages come from the NYSED Report Card Database, a single Microsoft Access file the State publishes annually (~370 MB zipped). Loader is the nysed_src module on our nycschools fork branch.

ESSA accountability status
2025 release · School-level overall tier (Local Support, TSI, ATSI, CSI), plus by-subgroup breakdown.
Chronic absenteeism
2025 release · % students absent ≥10% of enrolled days, by subgroup, EM + HS.
Per-pupil expenditures
2025 release · Federal + state-local + combined per-pupil dollars.
Teacher quality
2025 release · % inexperienced teachers, % inexperienced principals, % out of certification.
HS graduation rate
2025 release · 4-year / 5-year / 6-year cohort rates by subgroup.
College/Career/Civic Readiness
2025 release · HS accountability index weighting graduates by credential earned.

Other boundaries & geocoding

2010 NTA polygons
2010 vintage · Pulled from a community mirror — NYC retired the 2010 NTA dataset from Open Data when the 2020 NTA spec shipped, but our school-location data still uses 2010-era NTA names, so we need the 2010 polygons to draw the neighborhood pages.
NYC GeoSearch (runtime)
Current · geosearch.planninglabs.nyc.gov · NYC Planning Labs' address geocoder, called at runtime when a user submits an address to /find.

A note on years

NYC and NYSED publish each dataset on its own cadence. Some are current to the month (the staffing report is mid-2025-26), some are years stale (the DOE per-school portal scrape is roughly 2016). We don't try to harmonize: the most recent year available for each source is what we use, and the year is always listed alongside the data — on this page, on individual school pages (in the small "provenance" caption under each section), and on neighborhood pages.
